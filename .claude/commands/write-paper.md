Draft or continue writing a paper for the specified project.

Usage: /write-paper <project-name>

1. Read the project brief, status, and research notes
2. Check what sections exist in projects/<project-name>/paper/
3. Continue drafting the next section based on status.yaml next_steps
4. Write in LaTeX format following academic conventions
5. Update status.yaml with drafting progress
6. Commit with: paper(<project-name>): <description>
