### Task 1: Add Lemma A.1 — Verification Boundedness

**Purpose**: Prove that verification capability ν_t cannot grow unboundedly during self-training.

**What to prove**: If model M_t trains only on data generated and filtered by itself, then there exists ν_max such that ν_t ≤ ν_max for all t.

**Why this matters**: Theorem 1 needs this. Currently Theorem 1 assumes boundedness; Lemma A.1 will prove it.

#### Suggested Proof Approach

**Intuition**: Verification capability can only improve if the model sees harder problems than it initially could verify. But in self-training, the model only sees problems IT filtered as correct. So it never sees problems beyond its initial verification capability.

**Formal approach**:

1. **Setup**: Let V_t be the set of tasks model M_t can verify (Ver_M(x,y) ≥ threshold for x ∈ V_t).

2. **Key observation**: In self-training, training data D_t is generated by M_t and filtered by M_t. So D_t contains only tasks x where M_t successfully verified a solution.

3. **Inductive argument**:
   - Base case: V_0 = {tasks M_0 can verify}
   - Inductive step: M_{t+1} is trained on D_t ⊆ V_t. Training on tasks in V_t cannot teach verification beyond V_t (why? Need to formalize this — could use PAC learning bounds or information-theoretic argument).
   - Conclusion: V_{t+1} ⊆ V_t ∪ boundary(V_t) for some small boundary region.

4. **Boundedness**: If boundary is bounded (ε-expansion per iteration), then V_∞ ≤ V_0 + O(ε × iterations). If iterations converge, V_∞ is bounded.

```latex

\end{lemma}

[Step 3: Prove training on D_t cannot expand V_t beyond small boundary]
```

### Task 2: Characterize f(g_D) in Theorem 3

**Purpose**: Give explicit form or proven properties for function f(g_D) relating GV-gap to improvement ceiling.

**Current state**: Theorem 3 says "γ∞ ≤ ν₀ + f(g_D)" but f is never characterized. Lemma 4 provides "information-theoretic argument" but too vague.

**What's needed**: Either:
- **Option A** (preferred): Explicit functional form — f(g_D) = α exp(-β g_D) or f(g_D) = γ / (1 + g_D) or similar
- **Option B** (acceptable): Proven asymptotic properties — f(g_D) = O(1/g_D) or f(g_D) = Θ(exp(-g_D))
- **Option C** (minimum): Prove monotonicity and limits: f'(g_D) < 0, lim_{g_D → 0} f = f_max, lim_{g_D → ∞} f = 0

#### Suggested Approach

**Intuition**: When verification is much easier than generation (large g_D), self-improvement gains are small because the model can verify but struggles to generate. When verification and generation are similar (small g_D), larger gains possible.

**Formal approach**:

1. **Information-theoretic bound**: The maximum information gain about true solutions from self-verification is bounded by the mutual information I(Y; V(Y)) where Y are generated solutions and V is verification.

2. **Relate I to g_D**: When g_D is large (verification much easier), I is high (verification is informative). But generation improvement is limited by generation difficulty itself.

3. **Derive bound**: Use PAC learning sample complexity or information-theoretic rate-distortion theory to bound improvement as function of I and g_D.

4. **Result**: f(g_D) emerges from this analysis — could be exponential decay f(g_D) = ε_0 exp(-α g_D) where α depends on task structure.

### Task 5: Formalize Theorem 4 (Self-Play)

**Current issue**: Theorem 4's "objective outcome property" is defined informally by examples.

**What's needed**: Formal definition of "objective outcome" and rigorous proof of separation.

#### Formalize Definition 5

Current definition (informal):
> "A game G has the objective outcome property if game outcomes are efficiently computable independent of player capabilities."

Formal definition:
```latex
\begin{definition}[Objective Outcome Property]
\label{def:objective-outcome}
A two-player game $G$ has the \emph{objective outcome property} if there exists a computable function $W : S \to \{0, 1\}$ where $S$ is the set of terminal game states, such that:
\begin{enumerate}
\item $W(s) = 1$ iff player 1 wins in state $s$
\item $W$ is efficiently computable (polynomial time)
\item $W$ is independent of player strategies: for any players $\pi_1, \pi_2$, if game reaches state $s$, then $W(s)$ does not depend on $\pi_1$ or $\pi_2$
\end{enumerate}
\end{definition}
```

**Examples**:
- Chess: W(s) = 1 if white wins, 0 otherwise. Computable in O(1) (check board state). Independent of players.
- Go: Same.
- Math problems as game: W(s) = 1 if final answer is correct. Computable if verification is easy. Independent of solver.

#### Complete Theorem 4 Proof

```latex
\begin{theorem}[Self-Play Separation]
\label{thm:self-play-separation}
Consider two self-improvement processes:
\begin{enumerate}
\item Self-training without external verification (Theorem 1 setting)
\item Self-play on game $G$ with objective outcome property
\end{enumerate}

If $G$ has the objective outcome property, then self-play can achieve $\gamma_{\infty}^{SP} > \nu_0 + \epsilon$, exceeding the self-training bound.
\end{theorem}

\begin{proof}
\textbf{Case 1: Game with objective outcome property.}

Since $G$ has objective outcome property, there exists efficiently computable $W$ independent of player strategies (Definition~\ref{def:objective-outcome}).

Key observation: $W$ acts as an external verification oracle. Even though both players are copies of the same model, the game outcome $W(s)$ provides a signal that is:
\begin{itemize}
\item Independent of the players' current capabilities
\item Objective (not subject to the players' verification errors)
\item Informative (reveals which strategies are superior)
\end{itemize}

Formally, let $\pi_t$ be the policy at iteration $t$. Self-play generates trajectory $\tau \sim \pi_t \times \pi_t$ with outcome $W(\tau)$. Training on $\tau$ with reward $W(\tau)$ is equivalent to supervised learning with ground truth label $W$, not self-verification.

Therefore, the verification bound $\nu_0$ does not apply — we have access to an objective oracle through the game mechanics. By standard RL convergence results, policy $\pi_t$ can improve toward optimal $\pi^*$ where $\text{Gen}(\pi^*) > \nu_0 + \epsilon$.

\textbf{Case 2: Game without objective outcome property.}

If $G$ lacks objective outcomes (e.g., creative writing quality game), then outcomes depend on subjective judgment by the players themselves. This reduces to self-verification, subject to Theorem 1 bounds.

Therefore, self-play provides advantage only when objective outcome property holds.
\end{proof}
```
