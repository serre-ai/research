│

├─ Detection: Action similarity >0.8 for 10+ iterations
├─ Cost: $2.00-4.00 (3 trials, 50-iteration limit)
└─ Validates: Category 3.1 (Planning Progress Monitoring)

Test 3: CONTEXT DEGRADATION (Instance 49)
│
├─ Setup: 20 facts scattered across 32k tokens
├─ Task: "What is fact #7?" (position varies)
├─ Expected: >30% accuracy drop for mid-context facts
├─ Detection: Accuracy by position bucket
├─ Cost: $2.00-3.00 (10 positional trials)
└─ Validates: Category 4.3 (State Tracking Context Management)

PILOT DECISION POINT
