# optimal-cot-depth

**On the Optimal Depth of Chain-of-Thought: A Complexity-Theoretic Analysis**

Target: ICLR 2027

Resolves contradictions in CoT length literature by proving that optimal reasoning depth is determined by the computational complexity of the underlying task.
