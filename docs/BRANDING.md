# Research Lab Branding — Domain Candidates

Status: exploring. Current placeholder: darkreach.ai (owned, not committed to).

## Requirements
- `.ai` TLD
- Conveys authority as a research group (not a startup, not a product)
- Short, memorable, pronounceable
- Works on a paper byline: "Oddur Sigurdsson, {Name}"
- Works as a noun: "a paper from {Name}" / "the {Name} team"
- Icelandic touches welcome but not required
- Should feel like it belongs alongside: DeepMind, FAIR, Mila, EleutherAI, Anthropic

## Availability Check Template

Run for any candidate:
```bash
dig +short DOMAIN.ai
# Empty = likely available. IP result = taken.
```

Note: DNS check ≠ registration check. A domain with no DNS records might still be registered but not pointed anywhere. Check the registrar for definitive availability.

## Evaluation Criteria

Score each candidate 1-5:
- **Authority**: Does it sound like a research institution?
- **Memorability**: Will people remember it after hearing it once?
- **Pronounceability**: Can you say it in conversation without spelling it?
- **Brevity**: Is it short enough for a paper byline?
- **Distinctiveness**: Does it stand out from existing AI labs?
- **Versatility**: Does it work for papers, a website, and casual reference?
