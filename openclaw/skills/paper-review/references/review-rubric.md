# Review Rubric

Structured review framework for Deepwork Research outputs.

## Dimensions (1-5 scale)

### Correctness
- **5**: All claims proven or supported by evidence. No errors.
- **4**: Minor issues that don't affect main conclusions.
- **3**: Some unsubstantiated claims or minor errors in reasoning.
- **2**: Significant errors that affect conclusions.
- **1**: Fundamental correctness issues — results may be wrong.

### Novelty
- **5**: Genuinely new contribution — framework, technique, or finding.
- **4**: Meaningful advance over existing work.
- **3**: Incremental contribution with some new elements.
- **2**: Mostly rehashes existing approaches.
- **1**: No novel contribution identifiable.

### Rigor
- **5**: Methodology is bulletproof — pre-registered, well-controlled, statistically sound.
- **4**: Solid methodology with minor gaps.
- **3**: Adequate methodology but some questionable choices.
- **2**: Significant methodological concerns.
- **1**: Methodology fundamentally flawed.

### Completeness
- **5**: All promised work delivered. No loose ends.
- **4**: Nearly complete — minor items remaining.
- **3**: Partially complete — some sections need more work.
- **2**: Substantially incomplete.
- **1**: Only a skeleton or outline.

### Clarity
- **5**: Crystal clear — well-organized, precise, readable.
- **4**: Clear with minor rough spots.
- **3**: Understandable but could be clearer.
- **2**: Confusing in places — needs significant rewriting.
- **1**: Difficult to follow — structure and prose need overhaul.

## Verdict Mapping
- **ACCEPT**: Average 4.0+, no dimension below 3, no critical issues
- **REVISE**: Average 3.0-3.9, or any dimension below 3, with clear path to improvement
- **REJECT**: Average below 3.0, or any dimension at 1, or fundamental issues with direction

## Issue Severity
- **Critical**: Blocks acceptance. Must be fixed. (e.g., incorrect proof, wrong numbers)
- **Major**: Should be fixed before submission. (e.g., missing baseline, unclear methodology)
- **Minor**: Would improve the work. (e.g., better figure labels, clearer phrasing)
